## Introduction

Our product range includes advanced diagnostic equipment, patient monitoring systems, surgical instruments, and hospital furniture—carefully selected to meet international safety and performance standards. With a focus on innovation, durability, and precision, we help healthcare providers operate with confidence.
At MedEquip Solutions, we are committed to excellence, customer satisfaction, and long-term partnerships. Our experienced team is dedicated to offering expert guidance, dependable support, and solutions that improve healthcare outcomes.

## Installation

To use this tool, you'll need to have GitHub installed on your machine. Follow the instructions below to set up the environment and install the necessary dependencies.

1. Clone the repository:

    ```bash
    git clone https://github.com/peterBusto/project1-week2-peter.git
    ```

2. Navigate to the Project Folder

    ```bash
    cd medical-website`
    ```

3. Verify the File Structure

    ```bash
    Ensure the folder contains:
   - index.html
   - products.html
   - about.html
   - style.css
    ```
4. View the Website Locally

     ```bash
    1. Locate the project folder on your computer
    2. Double-click index.html
    3. The website will open in your default web browser
    ```
5. Navigate the Website
    ```bash 
    Use the top menu to visit:
    1. Home
    2. Products
    3. About Us
    ```
## Usage

This medical equipment website is designed to present company information and product offerings in a simple, professional, and user-friendly manner. Visitors can easily navigate between the Home, Products, and About Us pages using the top navigation menu.

The website can be used to showcase medical equipment, highlight product categories, and provide background information about the business. Administrators can update content, add new products, or modify styling by editing the HTML and CSS files, making the site easy to maintain and expand as business needs grow.
